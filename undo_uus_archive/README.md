# undo_uus_archive

This folder is a curator-level wrapper that groups the original files received from author Undo Uus.

## Provenance Statement

- The subfolders `_IMPERATIVE_Article/` and `_IMPERATIVE_Responses/` were provided as part of Undo Uus's private archive.
- Their internal file names and structure are preserved as received.
- This wrapper exists only to separate author-source materials from repository-added documentation and tooling.
