## Cite the Repository (Archive Edition)

Preferred citation format:

- *The Libertarian Imperative Archive*. Curated archival repository. Version/date: use release tag or commit hash and access date.

Example BibTeX:

```bibtex

  title        = {The Libertarian Imperative Archive},
  howpublished = {Git repository},
  note         = {Use commit hash and access date for reproducibility}
}
```
